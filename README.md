# Signup Page

This project is a simple and clean **Signup Page** created using **HTML**, **CSS**, and **Bootstrap 5**.  
It is designed with a responsive layout, modern UI, and a glass-effect container that works well on all screen sizes.

# Features

- Responsive signup form layout  
- Glassmorphism background container  
- Bootstrap 5 form styling  
- Input fields for:
  - First Name  
  - Last Name  
  - Email  
  - Phone Number  
  - Password  
  - Confirm Password  
- Clean hover and focus effects  
- Background image support  
- Redirect link for users who already have an account  

# Technologies Used

- HTML5 
- CSS3 
- Bootstrap 5.3
